Instagram-Bot
=============

Uploads imgur mirrors for instagram posts on [reddit](http://www.reddit.com/r/InstagramBot/).

### Requires:

- [praw](https://praw.readthedocs.org/en/latest/)
- [python-instagram](https://github.com/Instagram/python-instagram)
